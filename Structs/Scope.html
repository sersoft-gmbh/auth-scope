  • filter(_:)

    Filters the scope by calling isIncluded with each contained access range and only adding the ones for which it returns true to the result.

    Throws

    Any error thrown by isIncluded.

    Declaration

    Swift

    @inlinable
public func filter(_ isIncluded: (AccessRange) throws -> Bool) rethrows -> Scope

    Parameters

    isIncluded

    The closure called for each access range. Should return true if the access range should be included in the result, false otherwise.

    Return Value

    A filtered scope which only contains the access ranges for which isIncluded returned true.
